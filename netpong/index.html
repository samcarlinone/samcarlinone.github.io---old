Postmortem

This is another not so original concept, but it does have a few original technical features.

Perhaps the most interesting, at least in my opinion, is the automatic matchmaking. The game uses UDP multicast to find other people over LAN, and then initiate a connection. When this works it is pretty nifty, especially because no server is required. However it is glitchy at best, and you have to be on the same LAN so it’s obvious why it isn’t used more.

Another fun part of this project was the text. Since I was doing everything in OpenGL and I felt like a GUI library would be overkill, I ended up whipping up my own simple text shader with a bitmap font file. In this case, since I didn’t have to worry about any fancy formatting or wrapping, this solution worked great. Everything else is just triangles, including the ball.

This project is pretty small, but I really just wanted to see if I could get LAN discovery working.
